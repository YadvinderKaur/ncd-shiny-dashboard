NCD Global Health Indicator – Interactive Data Visualization Tool
Overview

The NCD Global Health Indicator Dashboard is an interactive Shiny application designed to explore and compare key non-communicable disease (NCD) indicators across countries and regions. The tool enables users to visually assess global and regional patterns in major public health risk factors using modern, interactive data visualizations.

The dashboard focuses on five core indicators:

Obesity rate (%)

Diabetes rate (%)

Hypertension rate (%)

Smoking rate (%)

Physical inactivity (%)

The application is intended for public health analysis, exploratory data visualization, and decision support.
Features

Global snapshot cards summarizing mean, median, and data coverage

Interactive world map with selectable indicators

Heat plot comparing countries across multiple indicators

2D scatter plot (Obesity vs selected indicator)

3D scatter plot (Obesity, Diabetes, Hypertension)

Regional boxplots for comparative analysis

Country profile view with indicator breakdown and table

Region and indicator filtering
